# Transferring Your Project

I have copied the final version of the **Melanin Intelligence Course** to your Desktop in a folder named `vines-connection`.

### 📁 How to Transfer
1.  **Copy the Folder**: Move the `vines-connection` folder from your Desktop to a USB drive or cloud storage.
2.  **Paste on New Computer**: Place the folder anywhere on the new computer.

### 🚀 How to Run (on the new computer)
1.  Open a Terminal or Command Prompt.
2.  Navigate to the folder (e.g., `cd Desktop/vines-connection`).
3.  Install dependencies (if not already there):
    ```bash
    npm install
    ```
4.  Start the course:
    ```bash
    npm run dev
    ```
5.  Open your browser to: `http://localhost:3000`
